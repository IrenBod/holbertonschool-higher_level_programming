# Python - Hello, World

## Description

The `python-hello_world` folder contains the initial tasks and projects aimed at introducing you to the Python programming language. These tasks cover basic Python concepts, including the use of the `print` function, data types, comments, and basic program structures.

## Content

- `0-run` : Script to execute Python code.
- `1-run_inline` : Script to execute Python code passed as a command-line argument.
- `2-print.py` : Python script that prints a string using the `print` function.

## Usage

You can run these scripts using Python in the command line. For example, to run the `2-print.py` file:

```bash
python3 2-print.py
